# AI Context & Data

This directory serves as the centralized hub for all AI-related inputs (instructions, prompts, knowledge bases) and outputs (analysis reports, generated artifacts).

## Directory Structure

*   **`instructions/`**: Inputs for AI models.
    *   **`datasets/`**: Synthetic and verified datasets (e.g., retrieval evaluation sets).
    *   **`experiments/`**: Historical and active experiment specifications and logs.
    *   **`knowledge/`**: Static analysis indices, probability graphs, and usage statistics.
    *   **`prompts/`**: Master system prompts used for forensic analysis and report generation.
*   **`reports/`**: Outputs generated by AI systems.
    *   Includes forensic analysis reports, integration logs, and documentation caches generated from source code.
