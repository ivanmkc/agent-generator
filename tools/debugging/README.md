# Debugging Tools

Scripts for inspecting traces, analyzing specific runs, and reproducing errors.

## Contents

- `debugging.py`: Main CLI for inspecting runs.
- `analyze_specific_run.py`: Analyzes a specific run directory.
- `extract_case_details.py`: Extracts details from a specific benchmark case.
- `inspect_trace_tool.py`: Deep dive into trace logs.
- `reproduce_one_liner_logic.py`: Verification script for one-liner generation.

## Usage

```bash
python tools/debugging/debugging.py --help
```
